# SPS Internship Assistant Demo

Demo website showcasing the SPS Internship assistant powered by IBM Watson Orchestrate chat.

This site integrates a dedicated SPS intern helper that answers questions about:

- Internship program and portal access
- IBM Cloud and IBM Partner Portal guidance
- Email/communication setup
- General intern FAQs

The live chat embed is configured in `index.html` with the current orchestration and agent IDs.
